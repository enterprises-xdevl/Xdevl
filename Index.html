⚠️ Minimal Install Note

This is currently a very minimal install. You must add packages and extensions based on your needs.

For example, to setup C++ development:

1) In the terminal:
   sudo apt install build-essential gdb

2) In vscode:
   Install the C++ extension

3) Have fun!

Versions already setup for certain development flows will be added in the future.

ℹ️ About Visual Studio Code

Visual Studio Code (VS Code) is a source-code editor made by Microsoft. Features include support for debugging, syntax highlighting, intelligent code completion, snippets, code refactoring, and embedded Git.

Users can change the theme, keyboard shortcuts, preferences, and install extensions that add functionality.

📱 Android Interface Guide

When using the graphical interface, use vscode just like normal. Here are specific gestures for the Android interface:

👆 Left Click
Tap with one finger.
🖱️ Move Mouse
Slide around one finger.
✌️ Right Click
Tap with two fingers.
🔍 Zoom
Pinch to zoom in or out.
✋ Pan
Press and hold, then slide one finger (useful when zoomed in).
↕️ Scroll
Slide two fingers up and down.
⌨️ Keyboard
Tap screen for icons to appear, then click the keyboard icon.
⚙️ Desktop Scaling
Find the service android notification and click settings. (Requires app restart).
💡 Tip: This is all easier to do on a tablet and with a stylus, but it can be done on a phone or using your finger as well.

📁 File Access

To access files from the rest of Android, there are many useful links in your home directory (/home/userland) to places like your Documents, Pictures, etc.

No need to import or export files.

❓ Frequently Asked Questions

📦 Puis-je installer Node.js ?

Oui, il suffit d'ouvrir un terminal et d'exécuter `sudo apt install nodejs npm`.

🔄 Comment mettre à jour VS Code ?

Les mises à jour sont gérées par le gestionnaire de paquets. Utilisez `sudo apt update && sudo apt upgrade code`.

📂 Où sont stockés mes fichiers ?

Tous vos fichiers sont dans `/home/userland`. Des liens symboliques vers vos dossiers Android (Documents, Images, etc.) sont disponibles.
